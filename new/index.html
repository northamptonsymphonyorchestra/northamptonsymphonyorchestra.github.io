NSO Style Guide / Test Card

READ THIS CAREFULLY!

This is a "test-card" page containing all the most common elements of a web-site to give an idea how a "real" page may look that also contains such elements. Nothing actually works!

Notice how the menu at the top is fixed so you never have to scroll around for it! There is also a standard footer containing a copyright notice an license information. Furthermore, this page is styled to work on all device forms, from a regular desktop to a mobile phone (try resizing your browser to see how it works as the width changes).

I've gone for a minimalist design since this has several advantages:

  • It looks clean.
  • It's easy to read.
  • It won't age.
  • It's simple to maintain.

I expect the actual page content to contain appropriate images and NSO branding. However, I've cobbled together a logo for the top menu so there is always some NSO branding visible on the page. This logo needs more work which I'll do later.

The next steps are for me to mock up each of the different pages so we get an impression of how the site hangs together.

Finally, each of the different types of element on a web page is described in more detail below.

Feedback most welcome!

Buttons

Buttons come in two basic flavors. The standard <button> element is plain, whereas the .button-primary button is vibrant and prominent. Button styles are applied to a number of appropriate form elements, but can also be arbitrarily attached to anchors with a .button class.

Forms

Forms are a huge pain, but hopefully these styles make it a bit easier. All inputs, select, and buttons are normalized for a common height cross-browser so inputs can be stacked or placed alongside each other.
